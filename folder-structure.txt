Key Components:

1. Authentication Flow:
   - JWT-based stateless authentication
   - Role-based access control (USER, ADMIN)
   - Secure password hashing with BCrypt
   - Custom UserDetails implementation

2. URL Management:
   - Short URL generation and redirection
   - Click tracking and analytics
   - URL validation and normalization
   - User-specific URL management

3. Analytics:
   - Real-time click event tracking
   - Date-based analytics
   - Per-URL and user-wide statistics
   - Click count aggregation

4. Security:
   - CSRF protection
   - CORS configuration
   - Input validation
   - Secure password handling
   - JWT token validation

5. Data Model:
   - User: Manages user accounts and authentication
   - UrlMapping: Stores URL shortening mappings and metadata
   - ClickEvent: Tracks detailed analytics data

6. API Endpoints:
   - /api/auth/** - Authentication endpoints
   - /api/url/** - URL management endpoints
   - /{shortUrl} - URL redirection endpoint
